About
=====

.. include:: ../README.rst


Features
========

* Execute arbitrary Python functions or methods on the CLI
* Local-command helper function

    * Captures & prints output
    * Allows interaction

* Namespaces
* Before/after hooks
* Aliases
* (Per-namespace-level) default tasks
* Color support
* Logging integration
* First-class library support


Installation
============

Invoke is best installed via `pip <http://pip-installer.org>`_::

    $ pip install invoke
